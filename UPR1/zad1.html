HTML

HTML(Hypertext Markup Language) is the set of markup symbols or codes inserted in file itnended for display on World Wibe browser page.The markup tells the Web browser how to display a Web page's words and images for the user.Each individual markup code is referred to as an element(but many people also refer to it as a tag).Some elements come in parts that indicate when some display is effect is to begin and when is to end.

CSS
